Two operators commute if and only if they have the same eigenvectors.

Total electronic orbital angular momentum of an atom: $\mathbf{L}=\sum_{i=1}^{n} \mathbf{L}_{i}$

Total electronic spin angular momentum of an atom: $\mathbf{S}=\sum_{i=1}^{n} \mathbf{S}_{i}$

Term: A set of equal-energy atomic states that arise from the same electron configuration and that have the same $\mathbf{L}$ value and the same $\mathbf{S}$ value constitutes an atomic term.

Total electronic angular momentum of an atom: $\mathbf{J} = \mathbf{L} + \mathbf{S}$

Level: The set of states that belong to the same term and that have the same value of $\mathbf{J}$ constitutes an atomic level.

The Hamiltonian operator for an n-electron atom without considering spin–orbit and other interactions (usually small except for atoms with high Z) : $\hat{H}=-\frac{\hbar^{2}}{2 m_{e}} \sum_{i=1}^{n} \nabla_{i}^{2}-\sum_{i=1}^{n} \frac{Z e^{2}}{4 \pi \varepsilon_{0} r_{i}}+\sum_{i=1}^{n-1} \sum_{j=i+1}^{n} \frac{e^{2}}{4 \pi \varepsilon_{0} r_{i j}}$
